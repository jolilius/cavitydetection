  <name>Task 1: Integrate explanation generation into run_experiment.py</name>
  <files>openevolve/run_experiment.py</files>
  <action>
Modify run_experiment.py to capture explanations for each iteration. The integration point is AFTER OpenEvolve completes and BEFORE consolidation.

**Flow:**
1. OpenEvolve finishes, writes best_program_info.json (unchanged)
2. Read best_program_info.json to extract best_program source code
3. Load explanation_prompt.txt
4. For each iteration in OpenEvolve's checkpoint (or just the best solution if only one available):
   - Call generate_explanation(evolved_code, baseline_code, config["llm"], prompt_text)
   - Store result in explanations dict: explanations[iteration_num] = explanation_text or None
5. Pass explanations dict to consolidate_experiment()
6. Write results.json with explanations included

**Key implementation details:**

- **OpenEvolve output parsing:** OpenEvolve writes checkpoints in openevolve_output/{prompt}/checkpoints/ or similar. Determine the structure by examining the output directory after a test run (or ask openevolve maintainers). For now, assume we can find best_program source in best_program_info.json.

- **Environment variable for toggling:** Add support for EXPLAIN_GENERATIONS=0 to skip explanation generation:
  ```python
  if os.environ.get("EXPLAIN_GENERATIONS", "1") == "0":
      explanations = {}  # skip explanation capture
      print("Explanation generation disabled via EXPLAIN_GENERATIONS=0")
  else:
      explanations = { ... generate explanations ... }
  ```

- **Baseline code:** Read from openevolve/initial_program.c (the seed program). Load it once at the start of the explanation capture loop.

- **Timing:** Explanation generation is expected to add ~1-2 seconds per iteration. Track and report it:
  ```python
  print(f"  Explanation generation: {explanation_elapsed:.2f}s")
  ```

- **Error handling:** If explanation_generator.py raises an exception or returns None, log a warning and continue. Do not block experiment results.

- **No modifications to evaluator.py:** The evaluator should be unchanged. Explanation generation happens outside the OpenEvolve loop.

**Code structure:**
```python
def main():
    # ... existing arg parsing and setup ...
    
    # Load baseline code and prompt
    with open(os.path.join(SCRIPT_DIR, "initial_program.c")) as f:
        baseline_code = f.read()
    with open(os.path.join(SCRIPT_DIR, "explanation_prompt.txt")) as f:
        explanation_prompt = f.read()
    
    # Run OpenEvolve as before
    subprocess.run([...], check=True)
    
    # NEW: Generate explanations if not disabled
    explanations = {}
    if os.environ.get("EXPLAIN_GENERATIONS", "1") != "0":
        try:
            from explanation_generator import generate_explanation
            explanations = _generate_explanations_for_experiment(
                output_dir, baseline_code, config["llm"], explanation_prompt
            )
            print(f"Generated {len(explanations)} explanations")
        except Exception as e:
            print(f"Warning: explanation generation failed: {e}")
            explanations = {}
    
    # Consolidate with explanations
    try:
        result = consolidate_experiment(
            output_dir=output_dir,
            program="cavitydetection",
            prompt_variant=args.prompt,
            baseline_accesses=128_862_705,
            explanations=explanations,  # NEW
        )
        # ... write results.json ...
    except Exception as e:
        print(f"Warning: consolidation failed: {e}")
```

Where _generate_explanations_for_experiment() is a helper that:
1. Reads best_program_info.json or checkpoint files
2. Extracts evolved source code
3. Calls generate_explanation() for each iteration
4. Returns dict[iteration_num, explanation_text]

```python
def get_explanations(df: pd.DataFrame) -> dict[int, dict[str, str]]:
    """
    Group explanations by iteration for analysis.
    Returns: {iteration: {"prompt_A": explanation_text, "prompt_B": explanation_text}}
    """
    result = {}
    for iteration in sorted(df['iteration'].unique()):
        iter_data = df[df['iteration'] == iteration]
        result[iteration] = {
            row['prompt']: row['explanation'] 
            for _, row in iter_data.iterrows() 
            if pd.notna(row['explanation'])
        }
    return result
```
