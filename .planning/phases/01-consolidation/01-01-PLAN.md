```python
def consolidate_experiment(
    output_dir: str,
    program: str = "cavitydetection",
    prompt_variant: str = None,
    baseline_accesses: int = 128_862_705,
) -> dict:
    """
    Consolidate one experiment's results from OpenEvolve output_dir into unified JSON.
    
    Args:
        output_dir: Path to openevolve_output/{prompt_name}/ directory
        program: Program name (default "cavitydetection")
        prompt_variant: Prompt name (auto-derived from output_dir basename if None)
        baseline_accesses: Reference baseline memory accesses
    
    Returns:
        Dictionary matching RESULTS_FORMAT.md schema; writes to {output_dir}/results.json
    """
```

**Implementation details:**

1. **Discover iteration history:** Walk `output_dir/` to find all iteration checkpoints or population snapshots. OpenEvolve outputs:
   - `best/best_program_info.json` (final best)
   - Population archive or checkpoint files (varies by OpenEvolve version)
   - Use `best_program_info.json` as fallback if per-iteration data unavailable (extracts: iteration, score, metrics)

2. **Extract per-iteration metrics from OpenEvolve output:**
   - If OpenEvolve writes per-iteration results: read directly
   - Fallback: Create synthetic iteration record from best result only (graceful degradation; Phase 1 can enhance later)
   - Parse `mem_score` or `combined_score` from metrics; convert to memory_accesses = baseline / mem_score
   - Calculate improvement_percent = (baseline - accesses) / baseline * 100

3. **Build consolidated structure:**
   - metadata: prompt from dirname, timestamp (current ISO 8601), llm_model from config (TBD: pass via consolidator), total_iterations (from OpenEvolve config), total_runtime (sum of iteration times if available, else estimate)
   - baseline_metrics: hardcoded split (reads = writes = baseline / 2, stores exact baseline)
   - iterations: sorted by iteration number

4. **Write output:** Serialize to `{output_dir}/results.json` (JSON, pretty-printed for readability)

5. **Error handling:** If best_program_info.json missing, log warning and return empty structure (allow partial consolidation for debugging)

**Do NOT:**
- Hardcode OpenEvolve file paths beyond standard openevolve_output structure
- Assume per-iteration data exists (fallback to best result)
- Write to cwd; always write to output_dir

```python
def load_results(filepath: str) -> pd.DataFrame:
    """
    Load consolidated results from a single JSON file into pandas DataFrame.
    
    Args:
        filepath: Path to results.json file
    
    Returns:
        DataFrame with columns:
          - iteration, memory_accesses, memory_reads, memory_writes, 
          - improvement_percent, iteration_runtime_seconds, mem_score
          - prompt (from metadata.prompt_variant), 
          - timestamp (from metadata), 
          - baseline_accesses (from metadata)
    
    Example:
        df = load_results('openevolve_output/baseline/results.json')
        df.plot(x='iteration', y='memory_accesses')  # convergence curve
    """
```

**Function 2: load_all_results(results_root: str = "openevolve_output") → pd.DataFrame**
```python
def load_all_results(results_root: str = "openevolve_output") -> pd.DataFrame:
    """
    Load all consolidated results from the results directory.
    
    Args:
        results_root: Root directory containing prompt subdirectories
    
    Returns:
        Combined DataFrame from all results.json files, indexed by prompt and iteration
    
    Example:
        df_all = load_all_results()
        best_by_prompt = df_all.groupby('prompt')['memory_accesses'].min()
        convergence = df_all.groupby('prompt').apply(lambda g: g[g['memory_accesses'] == g['memory_accesses'].min()]['iteration'].iloc[0])
    """
```

**Implementation details:**

1. **load_results:**
   - Read JSON file with `json.load()`
   - Extract metadata: prompt_variant, timestamp, baseline_accesses
   - Create DataFrame from iterations array
   - Add metadata columns (prompt, timestamp, baseline_accesses) to every row
   - Return sorted by iteration

2. **load_all_results:**
   - Walk `results_root/` for all `*/results.json` files
   - Load each with load_results() and concatenate
   - Return combined DataFrame sorted by (prompt, iteration)

3. **Error handling:**
   - If results.json missing: skip (warn if no results found at all)
   - If JSON invalid: log error and skip that file
   - If iterations array empty: create single-row DataFrame from metadata

4. **Column ordering:** Make iteration, prompt, memory_accesses, improvement_percent, mem_score the primary columns; metadata columns last

**Do NOT:**
- Require results to be in a specific location (accept any path)

```python
import json
import os
import tempfile
from consolidate_results import consolidate_experiment
from results_loader import load_results

def test_consolidation():
    with tempfile.TemporaryDirectory() as tmpdir:
        # Create synthetic OpenEvolve output
        best_dir = os.path.join(tmpdir, "best")
        os.makedirs(best_dir)
        
        # Write minimal best_program_info.json
        info = {
            "iteration": 25,
            "metrics": {
                "mem_score": 1.15,
                "combined_score": 1.15
            },
            "timestamp": "2026-05-13T14:00:00Z"
        }
        with open(os.path.join(best_dir, "best_program_info.json"), "w") as f:
            json.dump(info, f)
        
        # Consolidate
        result = consolidate_experiment(tmpdir, prompt_variant="test")
        
        # Verify structure
        assert "metadata" in result
        assert "baseline_metrics" in result
        assert "iterations" in result
        
        # Load into pandas
        results_json = os.path.join(tmpdir, "results.json")
        with open(results_json, "w") as f:
            json.dump(result, f)
        
        df = load_results(results_json)
        assert len(df) > 0
        assert "iteration" in df.columns
        assert "memory_accesses" in df.columns
        
        print("✓ All tests passed!")

if __name__ == "__main__":
```
